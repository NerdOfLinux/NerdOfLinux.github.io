Member Only Content Plugin

I wanted to create member only content that did not show a description on posts pages, so I finally decided to create a plugin.

All you need to do is install it, and then you can customize the message, sign in link, and whether or not to redirect the user to the post after loggin in. It works by checking if a post is in a category, and if it is, it displays the login message. It doesn't matter if a post is in multiple categories, if one is assigned as member only, it will be blocked.
