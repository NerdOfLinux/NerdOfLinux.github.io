Welcome to my first ever tutorial written on this site!!!! I will do my best to teach you the basics of HTML so you can start making basic-looking websites. Most websites have more than just HTML, but we'll talk about that in another lesson.

What you'll need

  • A web browser
  • A basic text editor(not Word, but something like notepad)
  • You could also use something like the Tryit Editor from w3schools(I use it a lot)
  • Time and patience

Let's start!

The nice thing about starting with just HTML is that it's 100% client side, meaning you won't need a server to host it. Simply make a .html file, and open it your browser! Be sure to use a simple text editor that won't format your text, or else you might have some problems.

HTML stands for Hyper Text Markup Language, and it is the language that the internet is written in. Sure, CSS, javascript, and a ton of other languages are also important, but HTML is still the core. The most important thing in HTML is tags, they look something like: 

 <tag> content </tag>

Tags basically tell the browser when something starts and when something ends. The tag you'll use most is probably the <p> tag, which is used to define a paragraph.

Now, onto the creating the site part! First, open your text editor, and copy and paste the following: 

    <html>
      <head>
        <title> This is the title </title>
      	<h1> This is the header </h1>
      </head>
      <body>
        This is the body of your site.
      </body>
    </html>
and open it in a file.

The <html> tag is at the top of more or less any website you go to, and is there so the browser knows that the HTML is starting. The <head> tag is short for header, and this is where I like to put the <title> tag, which sets the tab title, and the <h1> tag, which is the biggest heading text tag. All tags close with </tag>(replace tag with the actual tag name). The <body> tag is where the main content usually lives.

Past the above HTML code in a file called index.html, and open it in your browser. Then, replace the text in between the tags to whatever you want.

Congrats on making your first (basic) site!
